# Mode Selection

If the target Gem is not already explicit, start by asking which Gemini Gem mode the user wants:

1. General coding and architecture guidance
2. Frontend or web stack guidance
3. Python and backend guidance
4. Repo-maintenance and AI-tooling guidance
5. Custom bundle

Treat this as a small interactive decision step.
Do not dump all bundle ideas at once if the target is still ambiguous.
Pick the mode first, then provide only the relevant bundle and instructions.
